# 不为清单金字塔 (Not-To-Do List Pyramid)

基于“不为清单”原则的视觉化工具，通过明确边界，减少人生绕路，积累长期正向价值。

## 核心功能

- **金字塔视觉化**：直观展示你的边界。
- **分类管理**：深度工作、健康、社交、财务、其他。
- **能量黑洞审计**：复盘历史失败，提炼“不为”准则。
- **数据管理**：支持 JSON 备份/恢复，以及 Markdown 归档导出。
- **响应式设计**：适配桌面与移动端。

## 本地运行

1. **克隆仓库**：
   ```bash
   git clone <your-repo-url>
   cd nottodo-pyramid
   ```

2. **安装依赖**：
   ```bash
   npm install
   ```

3. **启动开发服务器**：
   ```bash
   npm run dev
   ```

4. **访问应用**：
   在浏览器中打开 `http://localhost:5173`。

## 技术栈

- React 19
- Vite 6
- Tailwind CSS 4
- Lucide React (图标)
- Motion (动画)
